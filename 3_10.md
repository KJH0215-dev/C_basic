### 에러에 대한 이해
# 컴파일 에러: 구문 분석 중 생긴 오류

# 실행 에러: 논리에 문제가 생겨 발생한 오류

# (+)디버깅: 프로그램이 올바른 순서로 실행되는지, 실행 중에 프로그램 내에서 사용된 수식의 값이 맞는지 확인

### C언어의 구성
기본 구성
#include <stdio.h>
int main(){

(함수)
return 0;
}
- #include: 전처리기
- <stdio.h>: 헤더 파일(라이브러리의 함수명, 리턴명 매개변수 제공)
- int: 리턴형
- main(): 진입점 함수 
- return 0: '0'을 붙이는 이유는 정상 종료의 의미를 포함한다.
- /**/: 여러 줄 주석 처리
- //: 1줄 주석 처리

### 변수 지정
- 변수란?: 값을 저장하기 위한 공간, 특정 메모리에 공간을 만들고 거기에 이름을 붙임
- 메모리의 이용: 컴퓨터 시스템은 값 저장을 위해 메모리 사용, 메모리는 한 바이트 단위로 저장
- 메모리의 구분: 메모리의 각 바이트를 구분하기 위한 주소(address) 사용
- 데이터형: 프로그램에서 사용되는 데이터의 종류(char, int, float etc)
- 서식 지정자:
- %d: 정수 출력
- %f: 실수 출력
- %c: 문자 출력
- (+): 함수에는 숫자 불가, 띄어 쓰기 불가 대신 '_'사용, 수업에서는 한글 사용 불가

## 느낀 점
일단 혼자서 할때 안된던 것들 대표적으로 3.14같은 소수는 출력이 안되길래 내가 뭘 잘못 했는지 몰랐는데 
오늘 학습으로 %f로 지정해야 하는걸 %d로 지정해놓은 것을 확인해서 선행도 선행이지만 일단 기본이 중요하단걸 깨달음
