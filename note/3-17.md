# C언어 공부

##  오늘 공부한 내용 (3-17)
- C의 입력과 출력

### printf()
- 문자열을 출력하는 기능 + 값을 서식에 맞춰 출력하는 기능 제공
- 예시 : printf("%d", a); / printf("%d %f", a, b);
- printf()의 서식 지정자
### printf()의 서식 지정자
- %d = 정수형
- %f = float형의 실수를 소수점으로 표현
- %x, %X = 16진수형(소문자, 대문자)
- %e = 실수를 지수형으로 표현
- %c = 문자 출력

### 문자 폭과 정밀도
- 문자 폭 지정 : %와 d 사이에 정수를 써준다. (%5d) / (%-5d)
- 문자 정밀도 지정 : %와 d 사이에 .과 정수를 적어준다. (%.5f)
- 문자 폭과 정밀도 모두 지정 : (%5.2f)

### scanf()
- 키보드로 입력한 값을 불러온다. 
- 특징 : 형식 문자열과 변수 이름을 지정, 변수 이름 앞에는 &를 써주어야 한다.
- 주의 사항 : 비쥬얼 스튜디오에선 호환? 문제 때문인지 scanf_s라 쳐줘야 괘찮더라 / 아니면 따로 위에 경고 안뜨게 지정을 해야함

### scanf()의 서식 지정자
- %d = 10진수
- %x = 16진수
- %i = 정수를 10진수, 8진수, 16진수로 입력
- %f = float형 실수형 
- %lf = double형 실수형
- %c =  문자형
- %g = 유효 숫자 이용할 때 사용
- () : 서식 지정자 사이 = 공백문자 표기
## 느낀 점
- 처음에 scanf 함수를 사용할 때 _s 안쳤다가 오류가 계속 나서 뭔가 해서 교수님께 여쭤보려 했으나 수업때 알려주셔서 다행이다.
